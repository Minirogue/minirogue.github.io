In addition to my academic work, I have a few personal projects:

HoloCanon: An Unofficial Star Wars Canon Tracker (Play Store, GitHub): This is an Android app I wrote to help myself keep track of what canonical Star Wars media I have and haven't consumed yet. I decided to flesh it out and release it for my fellow nerds. It has been received extremely well so far.

MuchToDo: This is another Android app that I'm working on. There may already be a lot of to-do list apps out there, but none of the apps I have found have been able to do exactly what I wanted. So my solution was to make my own. The primary features are task hierarchies and task contexts. More specifically, some tasks have sub-tasks that need to be completed, and sometimes those tasks have their own subtasks, which is a structure that most to-do list apps do not facilitate. Task contexts refers to date, time, location, mood, or anything else the user wishes to define as an appropriate or inappropriate time for a task. This way tasks can be filtered according to what is most relevant to the user at the time. While most of the core features have been coded, the app is still a work in progress, and there is no current timeline for its release.

Grid Diagrams

My current work involves using Markov chains to explore the trends of projected writhe of grid diagrams of knots as well as some more general combinatorial questions regarding grid diagrams.

Publications

A Symmetry Motivated Link Table - Using writhe data gathered from Monte Carlo sampling of links in the simple cubic lattice, we presented a way to choose a standard isotopy class for links with minimum crossing number 9 or less. This should aid in the communication of future research where link symmetries may be important.
